Privacy Policy for “Roblox Menu Restoration”

Effective Date: 09/04/2026

1.  “Roblox Menu Restoration” is a browser extension designed
    to modify the appearance and functionality of the Roblox website to
    restore previous menu features and improve user experience.

2.  Data Collection: This extension does not collect, store, or transmit
    any personal data.

-   No personal information is gathered
-   No browsing history is stored externally
-   No data is sent to any servers

All changes made by the extension happen locally within your browser.

3.  Permissions Usage: The extension may request access to certain
    websites in order to:

-   Modify page layout and interface elements
-   Restore or enhance menu functionality

These permissions are used solely for the extension’s core features and
nothing else.

4.  Third-Party Services: This extension does not use third-party
    analytics, tracking tools, or external services.

5.  Data Security: Since no user data is collected or transmitted, there
    is no risk of your personal data being exposed through this
